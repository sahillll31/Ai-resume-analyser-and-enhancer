# AI Resume Analyser and Enhancer

An AI-powered tool to analyze and enhance your resume, helping you match perfectly with job descriptions. Built using modern web technologies to provide an intuitive, secure, and blazing-fast experience.

## ✨ Features

- **AI Resume Analyzer**: Upload your resume, provide a job listing, and instantly get an in-depth ATS evaluation, score, and customized feedback.
- **AI Resume Enhancer**: Generate job-specific, ATS-optimized resume content in Markdown format, dynamically updated based on the provided job description and your past experience.
- **Robust Storage & History**: Keep all your resumes stored safely. Maintain a history of your evaluations, custom outputs, and AI-generated suggestions.
- **Modern UI**: Clean, responsive, and seamless interface utilizing React, Tailwind CSS, and shadcn/ui.
- **Easy Authentication**: Fast and reliable user authentication.

## ⚙️ Tech Stack

- **Frontend**: React, React Router v7, Tailwind CSS, shadcn/ui
- **Language**: TypeScript
- **Backend / Authentication / AI Layer**: [Puter.js](https://puter.com/)
- **Build Tool**: Vite

## 🚀 Quick Start

1. **Clone the repository:**
   ```bash
   git clone https://github.com/sahillll31/Ai-resume-analyser-and-enhancer.git
   cd Ai-resume-analyser-and-enhancer
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   ```

4. Open [http://localhost:5173](http://localhost:5173) in your browser to start building your perfect ATS-friendly resume!
